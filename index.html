Apple Intelligence and AI built into the editor.

Vibeit treats AI as a coding surface: inline previews, focused-cell patches, notebook context, and provider choices that fit each device.

On-device first when Apple Intelligence is available.

The default Apple Intelligence provider uses Apple's Foundation Models through the system language model. No API key is required when the device supports it.

How to use Vibeit packages.

Runtime installation is intentionally simple for users, while native wheel delivery remains a developer-controlled build step.

Open Packages.

Go to the Packages screen inside Vibeit and keep the default catalog URL, or paste the raw catalog URL from this page.

Refresh catalog.

The app downloads the JSON catalog, shows installable pure Python packages, and separates native entries from runtime installs.

Install packages.

Installable wheels are downloaded, verified with SHA-256, checked for unsafe files, and added to the embedded interpreter path.

Run notebooks.

Restart the kernel when needed, import the package, and continue working from the same notebook or script workspace.
